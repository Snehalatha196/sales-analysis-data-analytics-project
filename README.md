# sales-analysis-data-analytics-project
Sales Analysis Dashboard using Excel, SQL and Power BI

## Tools Used
• Excel – Data cleaning
• SQL – Data querying
• Power BI – Dashboard creation

## Project Steps
1. Cleaned sales dataset using Excel
2. Created SQL database and performed analysis queries
3. Built interactive dashboard in Power BI

## Dashboard Features
• Total Sales KPI
• Total Orders
• Sales by Product
• Sales by City
• Sales by Category
• Sales Trend
• Top Products

## Skills Demonstrated
Data Cleaning
Data Analysis
SQL Queries
Data Visualization
Business Intelligence
